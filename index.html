FDO2map

Resolve a FAIR Digital Object and open it in an interactive map dashboard.

Paste an RO-Crate persistent identifier from ROHub

Required only for datasets behind an authenticated proxy

  • Load RO-Crate from ROHub
  • Find ViewAction annotations
  • Resolve dataset URL
  • Open dashboard
